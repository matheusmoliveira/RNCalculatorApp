# React Native - calculatorApp

Getting started

1. Clone the repo:

```
git clone https://github.com/matheusmoliveira/calculatorApp.git

```

2. Create a new React Native app for each project

```
react-native init RNCalculatorApp
react-native init RNCalculatorApp
```
